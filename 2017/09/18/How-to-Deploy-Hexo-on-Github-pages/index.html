How to Deploy Hexo on Github pages

發表於 2017-09-18

第一篇文章不可免俗地來介紹這個 blog 怎麼建起來的，最陽春的那種 :)

主要有三個部分，

  1. Cloud9 設置免費的 Ubuntu 14.04 的
  2. 安裝與設定 Hexo
  3. 編輯完推上 Github Pages

之前在考慮 Pelican 、 Hugo 、 Hexo ，聽說了 Pelican 開發較不活躍且 render 速度慢， Hugo 配置較麻煩，就採用了 Hexo，希望有 Python 寫的靜態網站產生器同時又兼具美觀和易用性…這次架 Hexo 的感覺真的蠻快蠻方便的

Cloud9 簡易設置

Cloud9 自稱是雲端 IDE ，我都當作免洗的 Linux container(Ubuntu 14.04) 來用，我在上面做大部分的 web 開發與測試，往後會再寫一篇關於 c9 開發環境的設置，這邊先簡略的寫一下，依 Hexo 的需求，就是裝一個 Nodejs 的環境，有很多種裝法，我們這次用 Nodejs 官方教學直接用 apt-get 來裝
